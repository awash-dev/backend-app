# E-commerce Backend

This is the backend for an E-commerce application built with Node.js, Express, and MongoDB.

## Features

- User registration and login
- Profile update
- Product management
- Middleware for logging, CORS, and JSON parsing

## Prerequisites

- Node.js
- MongoDB
- Vercel CLI (for deployment)
